# iban
Generate random IBAN

Just run `make`

```shell
IBAN 1.0.0
Generate random IBAN

USAGE:
    iban [FLAGS] [OPTIONS]

FLAGS:
    -h, --help            Prints help information
    -p, --pretty-print    Pretty print the IBAN
    -V, --version         Prints version information

OPTIONS:
    -c, --country-code <COUNTRY CODE>    IBAN country code
```
